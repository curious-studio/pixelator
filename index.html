About this tool

I created this tool as part of a curriculum on the Elements and Principles of Design. It is intended to help design students explore the Elements of Line, Texture, and Value, as well as the Principles of Contrast, Pattern (Repetition / Rhythm), and Scale / Proportion.

This tool creates mosaics in the opus regulatum style, in which tiles or tessarae are placed according to a regular grid. Example pieces created with this tool can be found here. Tag or message me if you find this tool useful—I'd love to see what you make with it!
